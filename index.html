Controles

En este juego controlarás a Wenge, una chica Malawi que debe esquivar coches
Para moverla deberás usar los botones A y D para moverla lateralmente
A para moverla hacia la derecha
D para moverla hacia la izquierda
También en ciertos momentos podrás usar el ratón para comprar comprar en la tienda o selecionar texto En cierto momento del juego podrás comprar un dash, para usarlo
Q para la derecha E para la izquierda
